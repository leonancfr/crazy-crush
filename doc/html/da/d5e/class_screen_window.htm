Descrição Detalhada

Classe para criar uma janela dentro do jogo.

O tamanho mínimo da janela depende do tipo da mesma. Caso seja definido um valor menor que o mínimo, o valor mínimo será assumido.

Todos os elementos internos da janela possuem posição e índices (no ZOrder) relativos aos valores da janela. Estes ajustes só ocorrem no momento em que a mesmo é exibida
