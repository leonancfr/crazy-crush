Implementação de um botão circular.

A classe é instanciada com os seguintes valores padrões: Raio = 199 label: offset = (0,0) alinhamento = centralizado cor = (255,255,255,255) botão: cor normal = blue cor hover = blue_light

Observação: O raio do botão é largura/2
